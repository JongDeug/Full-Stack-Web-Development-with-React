script 태그를 마지막에 넣는 이유?
- 자바스크립트는 실행하는데 시간이 걸리므로 제일 마지막에 넣는 것이 좋음.

CSS
- align-items/content는 부모 요소의 자식들 전체를 묶어서 정렬해줌. 반면에 부모 요소의 자식 아이템들을 각각 다른 방식으로 정렬하고 싶다면
align-self를 사용하면 된다.


Bootstrap
- navbar-expand-? 는 축소되는 시기를 결정할 수 있다. 예를 들어 navbar-expand-sm( >= 576px ) 일 때 576px 미만 부터 축소 돼라는 뜻
- col-form-label 은 label의 text가 input의 중앙에 위치하게 끔 맞춰주는 역할을 한다.
- row를 주면 해당 viewport에 맞게 width가 100%가 됨.
- d-flex에서 만약 align-items-center을 해주지 않으면 이미지는 높이에 맞춰서 꽉차게 늘어남. img에만 align-self-center을 해주면 이미지는 
높이에 맞게 자신의 크기를 유지하면서 중앙에 배치되지만 그 옆 문구들은 중앙에 맞춰지지 않음.
- col 에는 flex-shrink를 관리하는 프로퍼티가 존재. 즉 상위 row의 flex-shrink: 0;을 없에기 때문에 (줄어드는게 가능해짐) 배치가 달라진다.
- tooltip과 popover은 javascript를 따로 넣어줘야한다.
- flex 일 때 m-auto를 해주면 저 멀리 보낼 수 있음.

HTML
- role 이라는 attribute를 사용해서 묶을 수 있다.
- radio box에서 input에 name을 동일하게 해야만 하나를 선택할 수 있게 된다.

Javascript
- NodeList vs Array 
1. NodeList는 javascript API가 아닌 브라우저에서 제공하는 API이다. (querySelectorAll, getElementByTagName 메서드를 사용했을 때 반환되는 것)
2. NodeList와 Array는 각각 고유한 속성과 메서드가 있고, 필요한 경우에 NodeList를 Array로 변환할 수 있다.
3. index.html script에서 map함수는 Array함수의 메서드이기 때문에 NodeList는 map을 사용할 수 없다. 그래서 [...NodeList] 문법을 사용하여 
Array로 변환 후 사용한다.
- classList contains 함수에서 클래스를 여러개 확인하고 싶을 경우 'bla', 'bla-bla' 처럼 쉼표로 나눠야함.

- attribute selector은 [href], [data-toggle="tooltip"] 처럼 []를 사용하면 된다.
